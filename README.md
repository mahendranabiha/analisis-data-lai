# Tutorial: Open Bike Sharing Dashboard

## by Localhost

### 1. Setup Environment - Anacoda
```
conda create --name data_analysis python=3.10
conda activate data_analysis
pip install -r requirements.txt
```

### 2. Run Streamlit App
```
streamlit run dashboard.py
```
